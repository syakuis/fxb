
### Introduce

모듈간의 상호작용을 위한 데이터를 관리하는 라이브러리입니다.

모듈의 데이터에는 어떤 모듈을 의존하고 있는 지 어떤 상태를 갖고 있는 지 알 수 있습니다.

데이터를 활용하여 모듈들은 유연하게 조립되고 다양하게 확장할 수 있습니다.
